#indian E-Commerce sales analysis

##Tools used
>Sql
>Excel
>Tableau

##Project overview
Analyzed Indian E-commerce orders data to find Sales trends,Profit patterns and top preforming categories.

##Key Findings
-Electronics has highest sales(Rs-1,65,267)
-Madhya Pradesh, Maharastra, Delhi are top 3 states
-Furniture has lowest profit margin
-Printers and Bookcases are most profitable sub-categories
-Top Customers:Yaanvi, Pooja, Abhishek
